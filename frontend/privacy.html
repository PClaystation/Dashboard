Privacy Notice

Continental ID uses cookies and local browser storage to keep you signed in and improve session reliability. This page explains what is stored and why.

What We Store

  • Session and refresh-token cookies set by the authentication API.
  • Local storage keys for account session state (for example token and user ID).
  • Optional local settings such as API or login popup override URLs.

How It Is Used

  • Authenticate requests to protected dashboard endpoints.
  • Maintain your signed-in session across page reloads.
  • Provide a stable login popup and account recovery flow.

Control

You can sign out anytime to clear active session state from this browser. Browser settings can also be used to clear site data and cookies.
