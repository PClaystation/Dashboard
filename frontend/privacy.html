Privacy Notice

Continental ID uses secure cookies and limited local browser storage to keep you signed in and remember interface preferences. This page explains what is stored and why.

What We Store

  • Secure session cookies set by the authentication API.
  • Local preference keys such as the active dashboard tab, cookie notice acknowledgement, and appearance toggles.
  • Access tokens are kept only in browser memory and are not persisted in local storage.

How It Is Used

  • Authenticate requests to protected dashboard endpoints.
  • Maintain your signed-in session across page reloads by refreshing from secure cookies.
  • Remember basic interface preferences on this device.
  • Track active sessions and account activity to help detect unauthorized access.

Control

You can sign out anytime to clear active session state from this browser. Browser settings can also be used to clear site data and cookies.
